💻 Sistema de Análise de Números em Java

Este é um projeto simples desenvolvido em Java com o objetivo de praticar lógica de programação, estruturas de repetição (while) e controle de fluxo (switch).

🚀 Funcionalidades

O sistema permite que o usuário:

Digite vários números pelo terminal
Encerre a entrada digitando 0
Escolha uma opção em um menu interativo para:
🔢 Contar quantos números foram digitados
➕ Calcular a soma dos números
📊 Calcular a média dos valores inseridos
🧠 Conceitos aplicados
Estrutura de repetição while
Estrutura de decisão if/else
Uso de switch case
Entrada de dados com Scanner
Operações matemáticas básicas
Validação para evitar divisão por zero
📌 Objetivo

Este projeto foi desenvolvido como parte dos estudos iniciais em Java, com foco em fortalecer a lógica de programação e preparar para desafios mais avançados.
